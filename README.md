# Plant Health & Insect Detection System

An IoT-based monitoring system to analyze plant health using real-time sensor data.

## Features
- Collects sensor data (temperature, humidity, soil moisture)
- Applies ETL techniques to clean and transform data
- Detects plant stress and insect activity
- Generates automated alerts

## Technologies Used
- Python
- IoT Sensors (simulated)

## Project Type
IoT & Data Processing
